Page Description:

This is a webpage using the DOM, where a specific actor's lines in a play can be highlighted with a single click. Other actors' lines should be un-highlighted simultaneously. The solution should be flexible enough to handle any number of actors and lines added to the page, all accomplished through JavaScript and DOM manipulation. This page avoided using page reloads and relying on third-party APIs like jQuery.

Here is the script to my play:

Joe: Why did the chicken cross the road?

Jake: I don't know, why?

Joe: I don't know, I'm asking you!

Chicken: I know! To get to the other side!.

Joe: Did that chicken just talk?.

Chicken: No.

Jake: Ok good! I though we were going crazy.

Chicken: Yeah... you guys are crazy.

Joe: We need to stop drinking Jack.

Jake: Agreed.
